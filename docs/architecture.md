# MiniAFL 架构概览（占位）

## 模块划分

- `core`: 调度、队列、能量策略、结果汇总
- `instrumentation`: 插装与 trace 采集适配
- `mutators`: 变异器与变异策略
- `targets`: 被测目标适配器（命令行/文件/网络）
- `utils`: 配置、日志、工具函数

## 开发路线示例

1. 实现 `CommandTarget` 以支持命令行目标
2. 实现 `BasicMutator` 并暴露插件接口
3. 在 `core` 中实现最小可运行调度器（调度->变异->执行->评估）
4. 添加插装适配以收集覆盖
